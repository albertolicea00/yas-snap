# Design System - Yet Another Store for Snap

This document outlines the visual identity and styling guidelines for **Yet Another Store for Snap**.

## Color Palette

The design is built around a modern, premium Dark Theme featuring a deep store base color combined with a signature Purple accent.

### Color Tokens

- **Store Base Color (Dark Gray)**:
  - Hex: `#1E1E2E`
  - HSL: `hsl(240, 21%, 15%)`
- **Accent Color (Purple)**:
  - Hex: `#822007`
  - HSL: `hsl(12, 91%, 27%)`
- **Subtle Highlight**:
  - Hex: `#8220071A` (10% opacity)
- **Text Primary**:
  - Hex: `#F8F8F2`
- **Text Secondary**:
  - Hex: `#A9B1D6`

## Visual Assets & Logo Concept

- **Base Structure**: Dark gray store icon outline or 3D block representing the storefront.
- **Accent Elements**: Highlights, borders, or light beams in **Purple**.
- **Inside the Store**: Stacked package cubes representing the packages managed by Snapcraft.
- **App Tag**: SNAP

## Typography

- **Headings**: Outfit or Inter (sans-serif)
- **Body Text**: Inter (sans-serif)
- **Monospace (for CLI output/commands)**: Fira Code or JetBrains Mono
